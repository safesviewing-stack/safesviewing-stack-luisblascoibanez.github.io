El Desarrollo

Este proyecto consiste en la creación de un brazo robótico hecho de forma casera utilizando impresión 3D. Todas las piezas se han fabricado una a una, lo que ha permitido adaptar el diseño poco a poco según se iba desarrollando. Para unir las diferentes partes se han utilizado tornillos, logrando una estructura resistente y fácil de desmontar si se necesita hacer algún cambio o mejora.

Para que el brazo pueda moverse, se ha añadido un sistema electrónico basado en Arduino. Este controla varios servomotores con diferentes niveles de fuerza, que permiten realizar los movimientos del brazo. Además, se incluye un módulo Bluetooth que permite manejarlo desde el móvil de forma sencilla. En una versión inicial, el brazo estaba montado sobre una base con ruedas omnidireccionales, lo que le daba movilidad, aunque esta parte no se llegó a documentar dentro del proyecto final.

El diseño del brazo destaca por su combinación de colores negro y amarillo anaranjado, que le da un aspecto llamativo y muy técnico. Esta elección ayuda también a diferenciar las distintas partes del mecanismo. En general, el proyecto refleja un enfoque práctico y experimental, donde tanto la estética como la funcionalidad han ido evolucionando a medida que se construía.
